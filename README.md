# 🏎️ F1info
F1info is a web app project focused on delivering Formula 1 information as:
- WDC (World Drivers' Championship)
- WCC (World Constructors' Championship)
- Head To Head comparison between drivers
- Information about latest session (location, weather, results)
- Information about every session since 2023 with results

# ⚙️ API's used:
- OpenF1 API for Formula 1 data
- Fetch API for fetching data from the OpenF1 API
